How to sync changes to your website:
1. Make changes in the tabs above
2. Click "💾 Sync to File" to download updated data.json
3. Replace your existing data.json file with the downloaded one
4. Refresh index.html to see the changes

Note: Changes are automatically reflected in index.html when you refresh the page, but the file sync ensures permanent storage.
